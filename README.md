# Smart Video Communications

A cloud-native video conferencing platform built with .NET 10 signaling, .NET Aspire 13 orchestration, Python AI services, WebRTC, and Azure AKS. Supports real-time audio/video, SFU integration, recording, transcription, translation, and AI-powered meeting insights.
